PROJECT OVERVIEW :-
In this course participants will create the required Digital Literacy Portfolio which assists in developing four digital competency requirements. The Digital Literacy portfolio consists of five components, including creating an infographic, creating an online professional presence, learning about teamwork, coding and how to communicate properly via email. The intent of the Digital Citizenship Portfolio is to ensure that individuals will be able to properly use digital technology and assist others in establishing a sense of digital safety and security in their online activities.

MODULE SUMMARIES
Module 1: Created a digital literacy infographic to spread awareness about safe and effective use of technology.
Module 2: Built a professional digital portfolio using platforms like GitHub and LinkedIn.
Module 3: Practiced coding on a platform and created a Google Form for collaboration and data collection.
Module 4: Learned professional email writing and responsible social media communication.
Module 5: Studied cybercrime types and created awareness through a case study and safety checklist.
